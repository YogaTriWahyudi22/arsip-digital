Aplikasi arsip digital ini untuk kelurahan koto di balai dimana bertujuan untuk mempermudah pengarsipan surat-surat pada
kelurahan tersebut.
serta mempermudah kepala lurah dalam melihat setiap tahun atau setiap bulannya surat mana saja yg banyak masuk ke kelurahan
untuk pengarsipan surat ini ada beberapa kategori surat spt : surat masuk, surat keluar, surat keterangan, surat keputusan
